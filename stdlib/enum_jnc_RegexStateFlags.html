bitflag enum jnc.RegexStateFlags

Overview

This enum describes available option flags for jnc.RegexState. More…

bitflag enum RegexStateFlags
{
    Incremental,
    Lexer,
};

Detailed Documentation

This enum describes available option flags for jnc.RegexState.

Enum Values

Incremental

Makes it possible to match input incrementally, i.e. chunk-by-chunk.

Lexer

When a final DFA state is reached, it’s condidered a successful match even when subsequent characters don’t match the regular expression. This behavior is typically required in lexers – hence, the name.
