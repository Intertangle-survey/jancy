bitflag enum io.ChildProcessEvents

Overview

bitflag enum ChildProcessEvents
{
    IoError,
    IncomingData,
    ReadBufferFull,
    WriteBufferEmpty,
    WriteBufferReady = WriteBufferEmpty,
    Eof,
    ProcessFinished,
    ProcessCrashed,
};

Detailed Documentation

Enum Values

IoError

An IO error occured on the file stream; detailed information about the error is available at io.FileStream.m_ioError.

IncomingData

More data is available on the file stream; this data could be fetched with io.FileStream.read method.

ReadBufferFull

File stream receive buffer is full; upcoming data may be dropped (depending on the underlying device represented by the file).

WriteBufferEmpty

File stream transmit buffer is empty.

WriteBufferReady

File stream transmit buffer is ready to accept more data; it’s OK to issue more io.FileStream.write requests.

Eof

End-of-file was reached on the stdin stream.
